# Job Tracker

A job application tracking tool built with React and TypeScript.  
This project allows users to manage job applications, track their status, and stay organized during the job search process.

---

## Features

- Add new job applications
- Edit existing applications
- Delete applications
- Filter by status (applied, interview, rejected, offer)
- Persistent storage using localStorage
- Modal-based form UI
- Smooth animations with Framer Motion

---

## Tech Stack

- React
- TypeScript
- Vite
- Framer Motion
